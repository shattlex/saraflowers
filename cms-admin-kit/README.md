# CMS Admin Kit (SARA FLOWERS)

Готовый переносимый комплект no-code админки (CRUD страниц, блоков, изображений) на `localStorage`.

## Что внутри
- `admin.html` — интерфейс админки
- `admin.css` — стили админки
- `admin.js` — логика CRUD страниц/блоков/медиа
- `content-store.js` — единое хранилище и дефолтные данные
- `site.js` — пример рендера публичной страницы
- `styles.css` — пример стилей публичной части
- `DATA_MODEL.md` — схема данных
- `INTEGRATION.md` — как подключить в другой проект

## Быстрый старт
1. Скопируй папку `cms-admin-kit` в новый проект.
2. Открой `admin.html` и проверь, что путь к `admin.js` корректный.
3. Подключи `content-store.js` в публичной части проекта и рендери данные из `loadContent()`.
4. Запусти проект на локальном HTTP-сервере (не через `file://`).

## Важно
- Хранение: `localStorage`, ключ `sara_flowers_cms_v1`.
- Это локальная CMS без авторизации и без бэкенда.
- Для production: вынеси хранение в API/БД + добавь auth.
